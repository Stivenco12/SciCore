Bienvenidos a SciCore

¿Qué es SciCore?

Aplicación para el cálculo de diferentes leyes físicas y químicas creada para facilitar el aprendizaje de los estudiantes en las ciencias básicas, donde los profesores podrán recomendar a sus estudiantes el uso de esta aplicación para realizar cálculos físicos de manera más rápida y sencilla.
