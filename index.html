Bienvenidos a SciCore

¿Que es SciCore?

Aplicacion para el calculo de diferentes leyes fisicas y quimicas creada para facilitar el aprendizaje de los estudiantes en las ciencias basicas de manera donde los profesores podran recomendar a sus estudiantes el uso de esta aplicacion para que puedan realizar sus calculos fisicos de manera mas rapida y sencilla.
